Contexto do Erro

Em qual página o erro aparece?

A

Página "Live" (/dashboard/live)

O erro ocorre quando abro a página de monitoramento ao vivo

B

Página "Câmeras" (/dashboard/cameras)

O erro aparece na página de gerenciamento de câmeras

C

Dashboard principal (/dashboard)

O erro ocorre logo ao entrar no dashboard

Análise técnica preliminar:

Se o erro é imediato, provavelmente há um useEffect ou componente que está tentando acessar o canvas/vídeo antes dos elementos estarem montados no DOM.

Solução provável:

Adicionar verificações de null e garantir que o vídeo está pronto (videoWidth > 0) antes de tentar desenhar no canvas.
