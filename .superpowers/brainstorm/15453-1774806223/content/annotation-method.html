Método de Anotação

Como o usuário vai marcar os objetos para treinamento?

A) Bounding Boxes Manuais

Usuário desenha retângulos ao redor dos objetos frame por frame

Pros: Maior precisão, controle total
Contras: Trabalhoso, demorado

B) Anotação Assistida por IA

YOLO sugere caixas, usuário ajusta/confirma

Pros: 10x mais rápido, bom para datasets grandes
Contras: Pode precisar de correções

C) Híbrido Inteligente (Recomendado)

IA anota automaticamente, usuário revisa só casos duvidosos

Pros: Melhor equilíbrio速度/precisão, escola bem
Contras: Um pouco mais complexo de implementar

🎯 Considerações importantes:

  • YOLOv8 precisa de ~100-1000 imagens por classe para bom desempenho
  • Anotação manual demora ~1-2 min por imagem
  • IA assistida reduz para ~10-20 seg por imagem
  • Interface precisa ser intuitiva - anotadores não são técnicos
